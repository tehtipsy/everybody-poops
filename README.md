# Everybody Poops

A lightweight web app for logging, viewing, editing, and exporting bowel movement records. Track timestamps, types, and notes — because everybody poops.

## Run the web app

```bash
python main.py
```

The app serves on `http://0.0.0.0:8000` (suitable for reverse proxying with nginx).
